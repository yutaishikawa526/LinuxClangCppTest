# 複数ファイル、ディレクトリでのインストール

## 概要
- インクルードパスはこのファイルのディレクトリ
- コンパイルは`./compile.sh`で実行し、`./out/result.out`に出力される。
- フォーマットは`./format.sh`で実行しする
